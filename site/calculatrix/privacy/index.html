Calculatrix

Privacy Policy

Last updated: 2026-05-30

Calculatrix is a calculator application. This page states the current privacy position for the public web distribution and for later store submissions.

Data collection

Calculatrix is intended to operate locally on the device. At this stage, the application is not designed to require account creation or direct submission of personal data to a Calculatrix-operated backend service.

Data sharing

Calculatrix is not intended to sell user data or share personal data with third parties for advertising purposes.

Device permissions

Any platform permissions requested by a published app build must be limited to what is necessary for the documented product behavior on that platform. Publication-specific declarations will be kept aligned with the shipped build.

Security and distribution

Public releases are intended to be distributed from official project channels. Release notes and store declarations should match the behavior of the shipped artifact.

Contact

For privacy questions related to Calculatrix, use the public project contact channels documented in the repository and release materials.
